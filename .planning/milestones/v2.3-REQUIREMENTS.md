## Out of Scope

| Feature | Reason |
|---------|--------|
| Include participant images in PDF | Increases complexity and file size significantly |
| Include raw participant responses | Report focuses on synthesis value, not raw data |
| Custom branding/logo options | Clean minimal design is the goal |
| Email PDF directly | Manual download and share for now |
| PDF password protection | Unnecessary complexity |
